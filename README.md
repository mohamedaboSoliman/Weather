# Interactive Weather App

**Live Demo:** [https://your-username.github.io/weather-app/](https://mohamedabosoliman.github.io/Weather/)

A modern and engaging web application for retrieving instant weather information for any city worldwide. The app features an interactive user interface with dynamic animations that change based on the current weather conditions, providing a unique and enjoyable user experience.

## Features

-   **Smart Search:** Search for any city using either Arabic or English names.
-   **Autocomplete Suggestions:** Get a list of suggested cities as you type to make searching faster and easier.
-   **City Dropdown:** Quickly select from a pre-defined list of major cities for instant results.
-   **Dynamic Animations:** Enjoy dynamic backgrounds and animations that adapt to the weather (sunny, cloudy, rainy, snowy).
-   **Responsive Design:** The application works perfectly on all devices, including mobile phones, tablets, and desktops.
-   **Comprehensive Information:** Displays temperature in Celsius, a weather description, and clear weather icons.

## How to Run the Project

To get a copy of the project up and running on your local machine, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mohamedaboSoliman/weather-app.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd weather-app
    ```
3.  **Open the project in your browser:**
    Simply open the `index.html` file directly in your preferred web browser (e.g., Google Chrome, Firefox).

## API Key

This project uses the OpenWeatherMap API. For the application to function correctly, you will need to obtain your own API key:

1.  Create a free account on [OpenWeatherMap](https://openweathermap.org/api).
2.  Get your API key from your account dashboard.
3.  Open the `script.js` file and replace `YOUR_API_KEY_HERE` with your actual key.

    ```javascript
    const apiKey = 'YOUR_API_KEY_HERE'; // Place your key here
    ```
